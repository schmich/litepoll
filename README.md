# litepoll

## Development

- [Install MongoDB](https://www.mongodb.org/downloads)
- [Install Redis](http://redis.io/download)
- [Install NodeJS](http://nodejs.org/)
- Clone repo: `git clone git@github.com:schmich/litepoll`
- Install dependencies: `npm install`
- Run everything
 - `mongod`
 - `redis-server`
 - `npm start`
